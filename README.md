# FinBot - Personal Finance Tracker

FinBot is an Android application designed to help you track and manage your personal finances with ease.  Built with Kotlin and modern Android development practices, FinBot provides an intuitive interface for monitoring your income, expenses, and financial goals.

## Features

-  **Visual Analytics** - Interactive charts and graphs powered by MPAndroidChart to visualize your spending patterns
-  **Income & Expense Tracking** - Record and categorize all your financial transactions
-  **Financial Goals** - Set and track your savings goals
-  **Data Persistence** - Automatic saving with JSON-based data storage using Gson
-  **Backup & Restore** - Keep your financial data safe with backup functionality
-  **Modern UI** - Clean and intuitive Material Design interface
-  **Smooth Navigation** - Seamless navigation between different sections using Android Navigation Component

## Technology Stack

- **Language**: Kotlin
- **UI Framework**: Android Jetpack (ViewBinding, Navigation, Fragment)
- **Charts**: MPAndroidChart v3.1.0
- **Data Handling**: Gson for JSON serialization
- **Async Operations**: Kotlin Coroutines
- **UI Components**: Material Design 3, FlexboxLayout
- **Build System**: Gradle with Kotlin DSL

## Requirements

- **Minimum SDK**: Android 10 (API 29)
- **Target SDK**: Android 13 (API 33)
- **Compile SDK**: Android 14 (API 35)
- **Java Version**: 11

## Getting Started

### Prerequisites

- Android Studio (latest version recommended)
- JDK 11 or higher
- Android SDK

### Installation

1. Clone the repository:
```bash
git clone https://github.com/Malindup2/FinBot.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the app on an emulator or physical device

### Build

```bash
./gradlew build
```

### Run Tests

```bash
./gradlew test
```
